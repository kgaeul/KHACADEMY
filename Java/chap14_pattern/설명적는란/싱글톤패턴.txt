싱글톤 패턴 (singleton pattern)

	객체지향 디자인 패턴 중 하나 특정 클래스가 오직 하나의 인스턴스(객체)만을 생성하고 그 인스턴스(객체)에 접근을 제공하는 패턴
	싱글톤 패턴을 사용하면 어떤 클래스의 인스턴스(객체)가 오직 하나만 존재하도록 보장
	이 인스턴스(객체)에 대한 접근은 어디서든지 가능
	리소스 관리, 로깅, 데이터 베이스 연결, 설정관리 사용
	
MVC 패턴(model view controller)

	1.Model = 데이터와 비즈니스 로직 처리
	2.View = 사용자 인터페이스
	3.Controller = 모델과 뷰사이의 상호작용
